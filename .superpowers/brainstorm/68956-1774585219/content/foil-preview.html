This is a CSS approximation. The real effect uses a Metal [[ stitchable ]] shader applied via SwiftUI's .colorEffect() — per-pixel color shifting with diamond patterns, noise texture, and sin/cos rainbow channels driven by device tilt offset.

Architecture: FoilShader.metal + DeviceMotionManager + .holographicCard() modifier
On iPhone: gyroscope drives the offset · On Mac: mouse position drives offset
