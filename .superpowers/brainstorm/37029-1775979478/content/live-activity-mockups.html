Design Notes

  • Subject colors match the app's subjectColor(for:) mapping
  • Countdown uses tabular nums for stable width (no jittering)
  • Context states: In Class (blue) → Ending Soon (orange, <5min) → Break (green) → Lunch (purple) → Completed (gray)
  • Progress bar is the only decoration — no icons, no room/teacher on lock screen (only in DI expanded)
  • Medium widget: timeline view with glowing dot for active class
  • Large widget: color-coded week grid, today's column highlighted, current period outlined
  • Explicit timeline updates (from PR #10 fix) — not periodic polling
